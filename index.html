Spacewarweb

This game is an homage to the classic game Spacewar created by the first hackers of the MIT AI Lab in 1962 using a PDP-1 computer. How the hell did it take me the same time using a macbook.

I got a bit lazy writing the menu so I hope you appreciate the compact information.

Controls:

WASD - Control Player 1
Arrow Keys - Control Player 2
P | ESC - Pause / Unpause
R - Restart

Caution! - If you haven't realized the sun pulls you towards itself
